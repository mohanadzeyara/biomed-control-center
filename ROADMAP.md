# BioMed Control Center Roadmap

## Phase 1 MVP

- Build a clean dashboard layout
- Add fake medical device status data
- Add a generated ECG signal endpoint
- Show ECG chart and heart-rate result
- Display simple alerts and recent analysis history
- Add learning notes for ECG and monitoring basics

## Phase 2

- Add CSV upload for ECG samples
- Add ECG filtering controls
- Add stronger R-peak detection
- Store analysis history in MongoDB
- Add threshold-based alert rules
- Add maintenance log editing

## Phase 3

- Add PDF report export
- Add WebSocket live ECG streaming
- Add medical image annotation tool
- Add authentication
- Add Docker setup
- Add deployment configs for frontend and backend

## Extra Ideas

- Add dark/light mode
- Add demo patient/session selector with fake IDs
- Add device reliability trend charts
- Add guided learning quizzes
- Add report templates for lab submissions
